# FinanceFlow — MVP v1.0

App de controle financeiro pessoal. React Native + Expo SDK 54.

## Funcionalidades

- Entradas de renda com múltiplas fontes
- Despesas variáveis por categoria
- Despesas fixas com controle de pagamento
- Metas e reserva de emergência
- Orçamento por categoria (Envelope Method)
- Score financeiro dinâmico (5 dimensões)
- Exportação CSV e PDF profissional
- Onboarding guiado 5 passos
- Alertas automáticos inteligentes
- Dark mode / Light mode / Sistema
- Projeção de saldo 90 dias

## Setup

```bash
pnpm install
npx expo start --clear
```

## Stack

React Native · Expo SDK 54 · NativeWind · Zustand · TanStack Query · AsyncStorage
